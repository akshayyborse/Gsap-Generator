# GSAP Animation Generator

A powerful, visual tool for generating GreenSock Animation Platform (GSAP) animations directly in your browser. Design complex animations with ease and export the code instantly.

## Features

- **Visual Controls**: Real-time sliders and inputs for:
  - **Transform**: X/Y translation, rotation, scale, skew.
  - **Timing**: Duration, delay, repeat, yoyo, and easing functions.
  - **Style**: Opacity, border radius.
- **Live Preview**: See your animations update instantly on various target elements (Box, Circle, Icons, Custom Images).
- **Code Export**: Automatically generates ready-to-use GSAP JavaScript code.
- **Custom Uploads**: Animate your own images.

## Technologies Used

- **HTML5 & CSS3**
- **Tailwind CSS** for styling.
- **GSAP (GreenSock Animation Platform)** for animation engine.
- **PrismJS** for syntax highlighting.
- **Lucide Icons** for UI elements.

## Getting Started

1. Clone the repository or download the files.
2. Open `index.html` in any modern web browser.
3. Start animating!
